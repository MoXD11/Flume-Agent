* A Flume Agent that will copy any files that will be created / moved to a specific directory. 
* It will copy it to you HDFS in the path you set in the config file. 
* You will need to change the source path, and the distination path to fit your environment.
* The agent will not delete the file after copying it. Instead, the file will be renamed to "x". completed ( "x" being the file name)
